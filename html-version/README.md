## ⚙️ Setup Instructions

### 1. Enable Contact Form (Required)

The inquiry form uses Web3Forms (free service). To activate it:

1. Go to https://web3forms.com
2. Sign up for free (no credit card required)
3. Create a new form and get your Access Key
4. Open `index.html` in a text editor
5. Find this line (around line 598):
   ```html
   <input type="hidden" name="access_key" value="YOUR_WEB3FORMS_ACCESS_KEY">
   ```
6. Replace `YOUR_WEB3FORMS_ACCESS_KEY` with your actual key
7. Save the file

**Form submissions will now be sent to your email automatically!**

### 2. Enable Tawk.to Chat Widget (Optional)

To activate the live chat:

1. Go to https://tawk.to
2. Sign up for free
3. Get your Property ID
4. Open `index.html` in a text editor
5. Find this line (around line 14):
   ```javascript
   s1.src='https://embed.tawk.to/YOUR_PROPERTY_ID/default';
   ```
6. Replace `YOUR_PROPERTY_ID` with your actual ID
7. Save the file

**Live chat will now work on your website!**

Open `styles.css` and find these color codes:
- Emerald green: `#10b981`
- Cyan blue: `#06b6d4`
- Orange: `#fb923c`

Open `index.html` and search for the text you want to change.

Each section is wrapped in `<section>` tags with IDs:
- `id="home"` - Hero

## 🔧 Troubleshooting

**Form not working?**
- Make sure you replaced `YOUR_WEB3FORMS_ACCESS_KEY` with actual key
- Check your Web3Forms dashboard for submissions

**Chat widget not showing?**
- Make sure you replaced `YOUR_PROPERTY_ID` with actual Tawk.to ID
- Wait a few seconds after page load

**Logo not showing?**
- Make sure `towntrend-logo.png` is in the same folder as index.html
- Check file name spelling (case-sensitive on some servers)
